# Eazy Compile GSM Sniffer Testing Environment

## in Kali Linux 2

## Start

```
chmod +x install.sh
./install.sh
```

## Install ARM Toolchain
```
/root/compile-arm-toolchain.sh
```

## Install osmocom-core
```
/root/compile-osmocom-core.sh
```

## Install osmocom-bb
```
/root/compile-osmocom-bb.sh
```
